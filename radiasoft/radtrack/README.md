### RadTrack Container (radiasoft/radtrack)

[RadTrack](https://github.com/radiasoft/radtrack) simplifies the use of
particle accelerator codes.

### To Run RadTrack

Inside container, you can start RadTrack with:

```bash
radia-run-radtrack
```
